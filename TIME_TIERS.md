## 怎么用

1. **从 T1 顺序往下扫**：每一档全部清掉再进下一档，建立模板自信。
2. **同档内随机抽**：参考 `pick.py`，目前还**不支持 tier 过滤**，要加我说。
3. **困在某档**：回到上一档随机抽 2-3 道当热身，让肌肉记忆复活。
4. **冲刺面试**：T1 + T2 + T3 共 53 道是面试**绝对核心**，5 分钟内能白板出来。

## 准确度声明

- "已完成" 严格定义：本地 `python xxx.py` 跑完所有 cases 全部 PASS。
- 估时基于"题面 + 标准套路"，**不计 debug 时间**。Python 语法卡壳 / 边界 / 用例 fail 会让实际耗时 ×2~3，尤其前两周。
- 自动检测方式：扫所有 `*.py` 看是否还含 `# TODO ... pass` 模板模式，未含的尝试 `python` 执行；只有 exit code = 0 算 ✓。
